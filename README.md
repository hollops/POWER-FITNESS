💪 Power Fitness Website
A modern, responsive gym website built using HTML, CSS, and basic JavaScript, designed to showcase a fitness brand with a clean and strong user interface.

🚀 Live Preview
[View Live Site](https://power-fitness-website.vercel.app)

📌 Project Overview
Power Fitness is a fully responsive gym website designed with a mobile-first approach. The goal of this project is to create a clean, modern, and visually engaging fitness website using core front-end technologies.

The design focuses on:
Strong visual hierarchy
Clean layout and spacing
Responsive design across devices

🎯 Features
✅ Responsive design (mobile, tablet, desktop)
✅ Modern hero section
✅ Training programs section (grid layout)
✅ Image gallery (4-column responsive grid)
✅ Trainers section
✅ Membership plans
✅ Testimonials section
✅ Contact section
✅ Mobile navigation menu (hamburger menu using basic JavaScript)

🛠️ Built With
HTML5
CSS3 (Flexbox & Grid)
JavaScript (for hamburger menu)
Responsive Design (Media Queries)
